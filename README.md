# Instructions To Use
## Download server.c and client.c in separate directories
## In server directory
```console
$ gcc server.c
$ ./a.out
```
## In client directory
```console
$ gcc client.c
$ ./a.out
```
## In client, to download file,
```console
client> get file1 file2 ...
```
# To exit, in client
```console
client> exit
```
